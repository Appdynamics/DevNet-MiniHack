Getting Started with AppDynamics Sample App

Welcome to the Sample App, a simple Fruit Stand inventory application. You'll use this app to go through some key AppDynamics concepts:


1. Business Transactions

Manage the inventory of a fruit vendor, and see how these end user operations are tracked as Business Transactions by AppDynamics.

2. Error Detection

Generate different kinds of application errors, and learn how these errors are tracked by AppDynamics.


AppDynamics Agents that monitor this application have already been configured for this Sample Application. The agents will send performance data to your AppDynamics Controller, which assembles, processes, and presents the data.
